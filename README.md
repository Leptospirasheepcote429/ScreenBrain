## 📌 What ScreenBrain Does

ScreenBrain is a screenshot knowledge base for macOS. It helps you search screenshots by what they show, not by file name.

Use it when you want to:
- find an old receipt screenshot
- locate a message or app screen you captured
- search text inside screenshots
- keep screenshots in one place
- use AI search with OpenAI or Ollama
- run local OCR for text lookup

It is built with SwiftUI and SwiftData, so it feels like a native Mac app

## 🖥️ What You Need

ScreenBrain is made for macOS.

You will need:
- a Mac running a recent version of macOS
- enough free storage for your screenshots and index data
- permission to let the app read your screenshot folder or chosen image folder

If you want AI search, you can also use:
- an OpenAI account and API key
- Ollama installed on your Mac for local models

## 🔍 How to Search

ScreenBrain looks at the content of screenshots, so you do not need to remember file names.

You can search for:
- text shown in a screenshot
- names of apps
- menu labels
- error messages
- dates and times seen on screen
- page titles or document text

Search results should show screenshots that match your query, then you can open the image and review it

If you want AI search, you can connect one of these options:

## 🧾 OCR Setup

OCR means optical character recognition. It lets the app read words inside screenshots.

Turn on OCR if you want to search for:
- chat messages
- web page text
- app labels
- document text
- error messages

For best results:
- use clear screenshots
- keep text large enough to read
- avoid blurry or dark images when possible

## 🛠️ Troubleshooting

### The app does not open
- Check that the download finished
- Open the app from Finder
- Allow it in the macOS security prompt

### Screenshots do not appear
- Make sure you picked the right folder
- Check that the folder has image files
- Wait for the first scan to finish

### Search returns no results
- Try a shorter search term
- Use text that appears in the screenshot
- Make sure OCR is turned on if you need text lookup

### OpenAI or Ollama does not work
- Check your API key or local model setup
- Make sure the app can reach the service
- Try a simple query first
